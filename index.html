Cloud Resume

A Cloud based web application project using various AWS services like Route 53, S3, CloudFront, Lambda, API Gateway, Amazon Certificate Manager, DynamoDB, AWS Organizations, etc. which showcases my resume with a visitor count. With the frontend riding on the shoulders of HTML, CSS and JavaScript and the backend leveraging AWS Serverless Application Model(SAM), most of the infrastructure is provisioned using infrastructure as Code(IaC) methodology.

  • Registered a domain name with a registrar and exported the zone file to Route 53 to use it as a domain service.
  • Provisioned AWS resource with AWS CDK(IaC) to store & host website in Amazon S3 and deliver it around the globe using CloudFront distribution.
  • Set-up a bash script and CI/CD to deploy the website to Amazon S3 & invalidate the Cloudfront Cache
  • Used API Gateway as a proxy to trigger AWS Lambda funtion which updates the website visitor count in DynamoDB.
  • Used GitHub Actions CI/CD to integrate and deploy the SAM infrastructure.
  • Authenticated GitHub to AWS by registering GitHub OIDC to federated identity in AWS Identity & Accessmanagement to facilitate GitHub Actions CI/CD.
